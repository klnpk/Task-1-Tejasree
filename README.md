# Project Title
Student Portfolio Website

## Description
This is a static Student portfolio website developed using HTML5 and CSS3 as part of DecodeLabs Internship Task 1.

The project showcases a modern student portfolio design with sections like Home, About, Skills, Projects, and Contact.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive Design
- Clean User Interface
- Navigation Bar
- Home Section
- About Section
- Skills Section
- Project Section
- Contact Section

## How to Run the Project

### Step 1: Open the GitHub Repository
Open the project repository from the GitHub link shared in LinkedIn.

### Step 2: Copy the Repository Link
Click on the green **Code** button and copy the repository link.

### Step 3: Open Git Bash
Open Git Bash on your computer. [Open Git Bash in desktop or downloads etc..]

### Step 4: Clone the Repository
Run the following command in Git Bash:

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### Step 5: Open the Project Folder
Move into the project folder using:

```bash
cd your-repository-name
```

### Step 6: Open the Project
Open the created project folder and double-click on:

```text
index.html
```

### Step 7: View the Website
The project will open in your default web browser.
